# Olympic Power Structure

## Overview
This project explores whether Olympic success is a fair competition or a reflection of global economic and geopolitical power.

## Research Question
Is Olympic success equally distributed across countries, or concentrated among a few dominant nations?

## Dataset
- Source: Olympic athlete dataset
- Level: athlete-event
- Transformed into: country-year dataset

## Methodology
1. Data cleaning and preprocessing
2. Deduplication of team events
3. Aggregation at country-year level
4. Exploratory data analysis

## Key Steps
- Avoid double-counting medals in team sports
- Compute country rankings per year
- Analyze concentration of medals

## Structure
- `notebooks/`: data preparation and analysis
- `data/`: raw and processed datasets

## How to run
Open the notebooks in Google Colab or Jupyter and run all cells.
